# Justificatifs PRO

Déposer ici les factures et reçus remboursables par l'employeur :

- Billets transport (vous uniquement)
- Factures hôtel (part pro ventilée)
- Inscription conférence
- Repas professionnels
- Transports locaux liés à la conférence

**Formats :** PDF, photo, email imprimé  
**Nommage :** `YYYY-MM-DD_nature_fournisseur.pdf`

Exemple : `2026-07-12_eurostar_adulte.pdf`
